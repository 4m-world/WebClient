5. {{ 'SPF' | translate }} (Recommended)

SPF is used to specify who is allowed to send email for the domain so we strongly recommend including ProtonMail in your SPF record. Please add the following TXT record into the DNS for {{ ctrl.domain.DomainName }}. This can typically be done in the control panel of your domain name registrar. {{ 'LEARN_MORE' | translate }}

Please add the following TXT record. Note, DNS records can take several hours to update.

{{ 'TYPE' | translate }} {{ 'HOST_NAME' | translate }} {{ 'VALUE_DATA_POINTS_TO' | translate }}
TXT @ v=spf1 include:_spf.protonmail.ch mx ~all
