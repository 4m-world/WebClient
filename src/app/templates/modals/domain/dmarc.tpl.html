7. DMARC ({{ 'OPTIONAL_ADVANCED' | translate }})

If you have set both SPF and DKIM, DMARC allows you to specify how other email services should deliver email for your domain if both SPF and DKIM checks have failed. This can make it harder for spammers pretending to be you but may also cause delivery issues if not done properly. {{ 'LEARN_MORE' | translate }}

Typical DMARC record. Note, you MUST change this before adding it to your DNS.

{{ 'TYPE' | translate }} {{ 'HOST_NAME' | translate }} {{ 'VALUE_DATA_POINTS_TO' | translate }}
TXT _dmarc v=DMARC1; p=none; rua=mailto:address@yourdomain.com

{{ 'TYPE' | translate }}:
TXT

{{ 'HOST_NAME' | translate }}:
_dmarc

{{ 'VALUE_DATA_POINTS_TO' | translate }}:
v=DMARC1; p=none; rua=mailto:address@yourdomain.com

p=none is report only mode so no effect on email delivery

address@yourdomain.com is where you will receive DMARC reports from other email services
